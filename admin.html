Logs & Export para Diagnóstico
Baixe os logs e cole no Claude para análise e correção automática
Como usar o relatório com o Claude
1. Baixe o relatório
Clique em "Logs para Claude" — gera um arquivo .txt compacto com todos os status, erros e logs de cada usuário.
2. Cole no Claude
Abra o chat do Claude, cole o conteúdo do arquivo e diga: "Analise esses logs e me diga o que está errado e como corrigir."
3. Só Erros (mais rápido)
Use "Só Erros" para um relatório só com usuários que têm problemas — arquivo menor, mais focado.
4. JSON / CSV para análise
Para análise detalhada em planilha ou por outro sistema, use o formato JSON ou CSV.
